# Notifikasi Otomatis via Telegram

Lihat menu **Notifikasi** → kartu "Kirim Ringkasan ke Telegram".
1) Buat bot via **@BotFather**, simpan token.
2) Vercel → **Env**: `TELEGRAM_BOT_TOKEN` → redeploy.
3) Isi **Chat ID** di aplikasi → **Kirim Sekarang**.

Untuk otomatis harian, lihat bagian Cron di dokumen ini (versi lengkap ada di kanvas).
